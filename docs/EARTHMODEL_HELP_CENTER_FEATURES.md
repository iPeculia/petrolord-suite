# EarthModel Pro Help Center - Feature Specification

## Core Features

### 1. Intelligent Search
*   **Real-time:** Results appear as you type.
*   **Multi-source:** Searches Articles, FAQs, Videos, and Glossary simultaneously.
*   **Weighted Ranking:** Prioritizes Title matches over Content matches.
*   **Suggestions:** Offers autocomplete suggestions based on common terms.

### 2. Navigation & Browsing
*   **Hierarchical:** Category -> Sub-topic -> Article structure.
*   **Breadcrumbs:** Easy navigation back to parent categories.
*   **Related Content:** Suggestions for "Read Next" at the bottom of articles.

### 3. Content Viewing
*   **Markdown Rendering:** Clean, readable text with semantic HTML.
*   **Rich Media:** Support for embedded images and video placeholders.
*   **Code Highlighting:** Syntax highlighting for Python/Scripting examples.
*   **Table of Contents:** Auto-generated links for long articles.

### 4. User Interaction
*   **Feedback:** "Was this helpful?" voting system.
*   **Sharing:** Copy link to clipboard or share via email.
*   **Printing:** Printer-friendly CSS for hard copies.
*   **Bookmarking:** (Planned for Phase 2) Save articles to personal list.

### 5. Accessibility
*   **Keyboard Navigation:** Fully navigable using Tab/Enter/Esc.
*   **Screen Reader:** ARIA labels on all interactive elements.
*   **Focus Management:** Focus trapped within modal when open.

### 6. Integration
*   **Context Aware:** Can be triggered to open specific articles based on the current active page.
*   **Non-Blocking:** Opens as a side sheet, allowing users to reference help while working in the main app.
