# Maintenance Schedule

## Weekly
*   **Review Feedback:** Check "Not Helpful" votes and user comments.
*   **Search Analytics:** Analyze "No Results" searches to identify content gaps.

## Monthly
*   **New Features:** Add articles for any features released in the monthly sprint.
*   **Link Check:** Automated check for broken internal/external links.

## Quarterly
*   **Full Audit:** Review top 20 most visited articles for accuracy.
*   **Asset Cleanup:** Remove unused images or resources from the storage bucket.
*   **Glossary Update:** Add new industry terms or acronyms introduced.

## Annually
*   **Structural Review:** Re-evaluate category hierarchy. Is it still logical?
*   **Video Refresh:** Re-record tutorials if UI has changed significantly.
