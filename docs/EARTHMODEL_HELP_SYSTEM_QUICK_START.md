# Quick Start: Help System

1.  **Open Help**: Click the `?` icon in the top-right header.
2.  **Search**: Type "Import LAS" in the search bar.
3.  **Select**: Click the top result "How to Import Well Logs".
4.  **Follow**: Read the steps. Use the "Copy Link" button to share with a colleague if needed.
5.  **Video**: If available, click the "Watch Video" tab within the article for a visual walkthrough.
6.  **Still Stuck?**: Click the "Chat" bubble in the bottom right to ask the AI assistant.
