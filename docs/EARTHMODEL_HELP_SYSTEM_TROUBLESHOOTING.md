# Help System Troubleshooting

| Issue | Possible Cause | Solution |
| :--- | :--- | :--- |
| **Search returns no results** | Typos or too specific keywords. | Try broader terms. Check internet connection. |
| **Video not playing** | Browser privacy settings or firewall. | Allow scripts from `cdn.petrolord.com` or checking VPN settings. |
| **Chatbot hallucinating** | Question is outside scope of documentation. | Verify info with official docs. Chatbot is an assistant, not an authority. |
| **Ticket submission fails** | Attachment too large. | Compress logs/screenshots to <10MB. |
