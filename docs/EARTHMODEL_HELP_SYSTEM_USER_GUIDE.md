# Help System User Guide

## 1. Introduction
The EarthModel Pro Help System is your comprehensive resource for navigating the platform, troubleshooting issues, and mastering advanced workflows. It is accessible via the **Help Icon (?)** in the top right navigation bar.

## 2. The Help Center
Clicking the help icon opens the Help Center sidebar.
*   **Search**: Use the search bar to find articles, videos, or forum threads. The search supports natural language.
*   **Categories**: Browse content organized by module (e.g., "Seismic", "Drilling") or type (e.g., "Tutorials", "References").
*   **Quick Links**: Access frequently used resources like "Keyboard Shortcuts" or "What's New".

## 3. Contextual Help
While working in specific modules (e.g., Well Planning), you may see small `?` icons next to complex parameters. Hovering over these provides instant definitions and usage tips without leaving the screen.
*   **Guided Tours**: When entering a new module for the first time, a guided tour may launch to highlight key features.

## 4. Support Channels
If documentation is insufficient:
*   **Chatbot**: Ask the AI assistant for immediate guidance.
*   **Live Chat**: Click "Chat with Support" to talk to a human agent (available 24/7 for Enterprise plans).
*   **Tickets**: Use the "Submit Ticket" form for detailed bug reports. Attach logs and screenshots for faster resolution.

## 5. Community Forum
Access the forum via the Help Center to:
*   Search for solutions posted by other users.
*   Post your own questions.
*   Vote on feature requests.

## 6. FAQ
**Q: Can I download the documentation?**
A: Yes, go to Help Center > Resources > Documentation to download PDF versions.

**Q: How do I report a critical bug?**
A: Use the Ticket System and mark priority as "Critical".
