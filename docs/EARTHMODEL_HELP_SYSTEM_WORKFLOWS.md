# Workflow: Resolving an Issue via Help System

**Goal**: User encounters an error during "Grid Generation" and needs a solution.

1.  **Initial Search**:
    *   User clicks **?** icon.
    *   User types "Grid generation error concave hull".
2.  **Review Results**:
    *   System displays 3 articles.
    *   User selects "Troubleshooting Grid Geometry Errors".
3.  **Apply Fix**:
    *   Article suggests "simplify boundary polygon".
    *   User follows link in article to "Polygon Editor".
4.  **Verification**:
    *   User retries Grid Generation.
    *   Success.
5.  **Feedback**:
    *   User clicks "Thumbs Up" on the help article.

# Workflow: Requesting a Feature

1.  **Access Forum**: Help Center > Community.
2.  **Search**: Check if feature already requested.
3.  **Post**: Click "New Topic" -> "Feature Request".
4.  **Detail**: Describe "Auto-export to PowerPoint".
5.  **Submit**: Community manager receives notification.
