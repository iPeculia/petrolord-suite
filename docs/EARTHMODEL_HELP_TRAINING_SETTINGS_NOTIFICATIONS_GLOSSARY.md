# Glossary

## Help System
*   **Article**: A single page of documentation.
*   **Contextual Help**: Info overlays triggered by UI interaction.
*   **Ticket**: A tracked record of a user issue or request.

## Training System
*   **LMS**: Learning Management System.
*   **Module**: A single unit of learning (video, text, or quiz).
*   **Path**: A curated sequence of courses.
*   **Badge**: Digital award for achievement.

## Settings System
*   **Workspace**: A shared environment for project collaboration.
*   **API Key**: A token used for programmatic access.
*   **2FA**: Two-Factor Authentication.

## Notification System
*   **Toast**: A transient popup message.
*   **Rule**: A logic condition (If X then Y) for generating alerts.
*   **Channel**: The medium of delivery (Email, SMS, In-App).
*   **Grouping**: Combining multiple similar events into one summary.
