# Metrics & KPIs

## Help System
*   **Deflection Rate**: % of searches that do NOT result in a ticket submission.
*   **Search Success**: % of searches where a user clicks a result.
*   **Article Rating**: Average user feedback score (1-5 stars).

## Training System
*   **Completion Rate**: % of enrolled users who finish a course.
*   **Skill Growth**: Delta between pre-test and post-test scores.
*   **Certification Ratio**: % of active users holding at least one valid certificate.

## Settings System
*   **Adoption**: % of users who have customized their profile or settings (vs defaults).
*   **Integration Health**: % of connected integrations reporting "Healthy" status.

## Notification System
*   **Open Rate**: % of notifications clicked by users.
*   **Action Rate**: % of notifications leading to a workflow action.
*   **Latency**: Time from event trigger to user delivery (Target < 500ms).
