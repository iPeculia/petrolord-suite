# Release Notes: User Support Suite v1.0

## New Features
*   **Unified Help Center**: Replaced old wiki with integrated search sidebar.
*   **Learning Management**: Built-in LMS for certification and training.
*   **Granular Settings**: Expanded control over notifications and performance.
*   **Smart Alerts**: Rule-based notification engine.

## Improvements
*   **Performance**: Settings load instantly from local cache.
*   **UI**: Consistent design across all support panels.
*   **Search**: Fuzzy search added to handle typos.

## Bug Fixes
*   Fixed issue where theme settings reset on logout.
*   Fixed broken links in documentation exports.
