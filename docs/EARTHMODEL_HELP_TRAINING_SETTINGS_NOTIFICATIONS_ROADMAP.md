# Product Roadmap

## Current Release (v4.0)
*   **Foundation**: Complete implementation of Help, Training, Settings, and Notification UI and logic.
*   **Integration**: Full connectivity with Supabase backend and existing EarthModel modules.

## Next Release (v4.1) - Q1 2026
*   **AI Recommendations**: "Users like you also learned..." suggestions in Training Hub.
*   **Mobile App**: Native push notifications and offline training access on iOS/Android.
*   **Advanced Analytics**: Deeper insights into team learning patterns for Enterprise admins.

## Future Releases (v4.2+) - H2 2026
*   **Adaptive Learning**: AI-generated quizzes based on user's weak areas.
*   **Social Learning**: Real-time study rooms and collaborative annotation on help docs.
*   **Workflow Automation**: Visual builder for complex notification-triggered workflows (IFTTT style).
