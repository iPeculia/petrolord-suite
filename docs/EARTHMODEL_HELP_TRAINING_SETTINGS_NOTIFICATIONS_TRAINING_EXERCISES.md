# Training Exercises

## Exercise 1: Help Hunter
*   **Task**: Find the article on "Stochastic Facies Modeling" and bookmark it.
*   **Goal**: Familiarize with search and bookmarking features.

## Exercise 2: First Steps
*   **Task**: Enroll in "EarthModel Essentials" and complete the first module quiz.
*   **Goal**: Understand the LMS flow and progress tracking.

## Exercise 3: Workspace Wizard
*   **Task**: Change the theme to "Dark", set units to "Metric", and configure the sidebar to "Compact".
*   **Goal**: Learn personalization options.

## Exercise 4: The Watchman
*   **Task**: Create a notification rule: "Alert me via SMS if Porosity > 0.25 in Well-X".
*   **Goal**: Master the Rules Engine.

## Exercise 5: The Connector
*   **Task**: Generate a Read-Only API Key and test it using `curl` or Postman.
*   **Goal**: Understand developer settings.

## Exercise 6: The Master
*   **Task**: Complete the "Advanced Configuration" certification exam.
*   **Goal**: Validate comprehensive knowledge.
