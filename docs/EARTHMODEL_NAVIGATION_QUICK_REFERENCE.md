# EarthModel Pro: Navigation Guide

## Global Navigation

### Back to Dashboard
*   **Button:** Located at the top-left of the header (Left Arrow icon).
*   **Shortcut:** Press `Alt + B`.
*   **Action:** Returns you to the main **Geoscience Analytics** dashboard.

### Sidebar Navigation
*   **Toggle:** Click the "Menu" icon (hamburger) on mobile or use the collapse button on desktop.
*   **Modules:** Switch between Structural, Property, and Facies modeling using the sidebar buttons.

## Unsaved Changes Protection
The system monitors your work for unsaved changes (e.g., a well form filled but not submitted).

**If you try to navigate away with unsaved changes:**
1.  A dialog will appear: **"Unsaved Changes"**.
2.  **Save & Exit:** Saves your work and then navigates.
3.  **Discard:** Discards changes and navigates immediately.
4.  **Cancel:** Stays on the current page.
