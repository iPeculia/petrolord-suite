# Quick Start: Object Modeling

**Goal**: Create a simple meandering channel system in 5 minutes.

1.  **Launch App**: Open EarthModel Pro and select **Object Modeling**.
2.  **Create Channel**:
    *   Click **Channel** card.
    *   Set **Width** = 300m, **Thickness** = 15m.
    *   Set **Sinuosity** = 1.4.
    *   Set **Azimuth** = 45Â° (NE).
3.  **Visualize**:
    *   Click **Open 3D Editor**.
    *   You see the channel in the 3D box.
4.  **Place**:
    *   Drag the channel to the center of the grid.
    *   Use the **Placement Interface** to fine-tune Z depth.
5.  **Save**:
    *   Click **Save Object**. Name it "Channel_Demo_01".
6.  **Done**: You have created a geological object ready for property population.
