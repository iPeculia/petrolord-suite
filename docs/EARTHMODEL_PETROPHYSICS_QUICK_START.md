# Quick Start: Petrophysics

**Goal**: Create a Porosity vs. Permeability transform.

1.  **Launch App**: Open EarthModel Pro and select **Petrophysics**.
2.  **Open Cross-Plot**: Click **Cross-Plot Analysis** in the Hub.
3.  **Setup Axes**:
    *   X-Axis: Select `Porosity`.
    *   Y-Axis: Select `Permeability`.
    *   Scale: Set Y to `Log`.
4.  **Identify Trend**:
    *   Observe the scatter of points.
    *   Notice the linear trend on the semi-log plot.
5.  **Export**:
    *   Click **Export** to save the data points or the chart image.
