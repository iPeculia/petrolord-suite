# Quick Start: Settings System

1.  **Open**: Click the **Gear Icon**.
2.  **Theme**: Under **Display**, select "Dark Mode".
3.  **Units**: Under **General**, verify "Project Units" is set to your preference (e.g., Metric).
4.  **Profile**: Go to **Profile**, upload an Avatar image.
5.  **Save**: Changes are often auto-saved, or click "Apply Changes" if prompted.
