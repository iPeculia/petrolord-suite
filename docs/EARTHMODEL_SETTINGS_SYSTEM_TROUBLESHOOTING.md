# Settings Troubleshooting

| Issue | Possible Cause | Solution |
| :--- | :--- | :--- |
| **Changes revert after refresh** | Database sync failure. | Check internet. Wait for "Saved" indicator before closing. |
| **Dark mode glitching** | Browser cache issue. | Clear browser cache/cookies. |
| **API Key not working** | Expired or scope mismatch. | Generate new key with correct scopes. |
