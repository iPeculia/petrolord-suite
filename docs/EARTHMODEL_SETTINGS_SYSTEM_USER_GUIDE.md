# Settings System User Guide

## 1. Overview
The Settings Panel is the control center for your EarthModel Pro experience. Access it via the **Gear Icon** in the header.

## 2. Key Configurations
### General
*   **Language**: Switch interface language.
*   **Units**: Set project default to Metric (m, kg/mÂ³) or Field (ft, ppg). Note: This can be overridden per project.

### Display
*   **Theme**: Toggle Dark Mode (recommended for interpretation) or Light Mode.
*   **Density**: "Compact" allows more data rows on screen; "Comfortable" adds whitespace.

### Performance
*   **Graphics**: Adjust 3D rendering quality. Lower settings improve performance on laptops.
*   **Cache**: Clear local data cache to free up disk space.

### Integrations
*   **Connections**: Add credentials for external services (e.g., Salesforce, OSDU).
*   **Sync Frequency**: Define how often data synchronizes (Real-time, Hourly, Daily).

### Notification Preferences
*   **Channels**: Choose where you want to receive alerts (Email, SMS, In-App).
*   **Categories**: Toggle specific alert types (e.g., turn off "Social" but keep "System Critical").

## 3. API Management
For developers:
*   Generate Personal Access Tokens (PATs) for scripting.
*   View rate limits and usage history.
*   Rotate keys for security.
