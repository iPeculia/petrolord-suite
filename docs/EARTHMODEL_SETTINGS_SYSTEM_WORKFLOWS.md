# Workflow: configuring a New Project Team

**Goal**: Set up a shared workspace with specific settings.

1.  **Create Workspace**: Settings > Workspaces > New.
2.  **Define Defaults**:
    *   Units: Field (Imperial).
    *   CRS: EPSG:4326.
3.  **Invite Team**:
    *   Add email addresses.
    *   Assign roles (Editor, Viewer, Admin).
4.  **Permissions**:
    *   Disable "Delete Data" for Viewers.
    *   Enable "Export" for Editors.
5.  **Save**: Apply settings. All team members receive invites.
