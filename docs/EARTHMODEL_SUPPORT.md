# Support & Help

## Getting Help
If you encounter issues or have questions about EarthModel Pro v4.0.0:

1.  **In-App Help**: Press `F1` anywhere in the application to open the context-sensitive Help Center.
2.  **Documentation**: Browse the full guides in the `docs/` folder or the Legal > Documentation page.
3.  **Community**: Join our user forum (link in Dashboard) to discuss workflows.

## Reporting Issues
Please report bugs to the support team with the following info:
*   Browser Version
*   Steps to reproduce
*   Screenshots (if applicable)
*   Console logs (Press F12 -> Console)
