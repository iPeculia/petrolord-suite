# Test Case Documentation

## TC-001: Help Center Search
*   **Pre-condition**: Help Center is open.
*   **Input**: String "Well Planning"
*   **Expected Output**: List of articles containing "Well" or "Planning".
*   **Priority**: High

## TC-002: Theme Switch
*   **Pre-condition**: Settings Panel open.
*   **Action**: Select "Light Mode".
*   **Expected Output**: Background color becomes white (#ffffff), Text becomes black.
*   **Priority**: Medium

## TC-003: Course Enrollment
*   **Pre-condition**: User logged in, Training Hub open.
*   **Action**: Click "Start Course".
*   **Expected Output**: User redirected to Course Player; Course added to "My Learning".
*   **Priority**: High
