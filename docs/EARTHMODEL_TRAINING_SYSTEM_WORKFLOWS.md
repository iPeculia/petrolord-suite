# Workflow: New Hire Onboarding

**Goal**: Complete the "Basics of EarthModel" path.

1.  **Notification**: User receives "Assigned Training" alert.
2.  **Access**: Click link to Training Hub.
3.  **Module 1**: Watch "Navigation Overview" (10 mins).
4.  **Module 2**: Read "Data Import Standards".
5.  **Exercise**: Download sample `.LAS` file and import it into the app.
6.  **Quiz**: Pass "Data Management Quiz" (Score 5/5).
7.  **Completion**: Badge "Data Handler" unlocked. Profile updated.
