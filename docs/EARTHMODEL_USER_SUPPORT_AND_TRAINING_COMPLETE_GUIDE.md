# EarthModel Pro: Complete User Support & Training Guide

## 1. Introduction
This document serves as the master manual for all support, configuration, and learning features within EarthModel Pro.

## 2. The Ecosystem
EarthModel Pro is not just a tool but a platform. Surrounding the core technical modules (Geoscience, Reservoir, etc.) are four pillar systems designed to support the user:
1.  **Help System**: Immediate assistance.
2.  **Training System**: Long-term skill growth.
3.  **Settings System**: Personalization and configuration.
4.  **Notification System**: Situational awareness.

## 3. Using the Help System
*   **Access**: Top navigation bar `?` icon.
*   **Capabilities**: Search docs, watch videos, chat with AI, contact support.
*   **Best Practice**: Use Contextual Help (hover hints) for quick definition checks during data entry.

## 4. Training & Certification
*   **Access**: Settings > Training Hub.
*   **Flow**: Take an assessment -> Enrol in a Path -> Watch Videos -> Pass Quizzes -> Get Certified.
*   **Benefit**: Certified users get early access to Beta features.

## 5. Configuration (Settings)
*   **Access**: Gear Icon.
*   **Critical Setup**: Ensure your Timezone and Unit System match your project requirements to avoid data conversion errors.
*   **Integrations**: Connect your corporate data stores here.

## 6. Notifications
*   **Access**: Bell Icon.
*   **Philosophy**: "Signal, not noise." Use filters to focus on what matters. Set up custom Rules to automate monitoring of your assets.

## 7. Getting Support
*   **Level 1**: Search Documentation / AI Chat.
*   **Level 2**: Community Forum.
*   **Level 3**: Support Ticket / Live Chat.

## 8. Glossary
*   **LMS**: Learning Management System.
*   **PAT**: Personal Access Token (for API).
*   **Toast**: Small pop-up notification at the edge of the screen.
*   **Contextual Help**: Help content relevant to the specific screen currently viewed.
