# EarthModel Pro: Well Management Quick Reference

## Well Creation Workflow

### Step 1: Navigate
Open **EarthModel Pro** and locate the **Wells** panel in the left sidebar.

### Step 2: Add Well
1.  Click the **+ Add** button in the Wells panel header.
2.  A dialog will appear.

### Step 3: Enter Details
*   **Name (Required):** Must be unique within the project.
*   **Coordinates (Optional):** Enter X/Y surface coordinates.
*   **Type:** Select Vertical, Deviated, or Horizontal.

### Step 4: Create
Click **Create Well**. A success message "Well Created" will appear, and the well will be added to your list.

## Common Tasks

| Task | Action |
| :--- | :--- |
| **Search Wells** | Use the search bar at the top of the Wells panel. |
| **Edit Well** | Click a well in the list to open its properties. |
| **Import Wells** | Use the "Import CSV" button for bulk data. |
| **Delete Well** | Select a well and click the Trash icon (requires confirmation). |

## Keyboard Shortcuts
| Shortcut | Action |
| :--- | :--- |
| `Ctrl` + `S` | Save current changes |
| `Alt` + `B` | Return to Dashboard |
| `F1` | Open Help Center |

## Common Errors & Fixes
*   **"Authentication Required":** Your session expired. Please log in again.
*   **"Duplicate Well Name":** A well with this name already exists. Choose another.
*   **"Violates Security Policy":** Ensure you are the owner of the project.
