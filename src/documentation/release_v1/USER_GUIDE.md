# User Guide

## Getting Started
1.  **Log In**: Access the portal and sign in with your corporate credentials or email.
2.  **Dashboard**: You will land on the main dashboard showing your recent projects.
3.  **Create Project**: Click "New Project", enter a name and CRS (Coordinate Reference System).

## Modules

### 3D Window
*   **Navigation**:
    *   Left Click + Drag: Rotate
    *   Right Click + Drag: Pan
    *   Scroll: Zoom
*   **Objects**: Toggle visibility of Wells, Horizons, and Faults in the Tree Panel (left).

### Well Correlation
1.  Select the **Correlation** tab.
2.  Click "Select Wells" to choose which wells to compare.
3.  Drag tops up/down to adjust.
4.  Use **AI Auto-Picker** (wand icon) to suggest missing picks.

### Importing Data
1.  Go to **Project > Import**.
2.  Select data type (LAS, SEG-Y, CSV).
3.  Drag and drop files.
4.  Map columns if headers are not standard.
5.  Click "Import" and wait for processing.

## Collaboration
*   **Presence**: See avatars of colleagues in the top right.
*   **Chat**: Open the right sidebar to chat with project members.
*   **Sharing**: All changes (picks, models) are saved instantly and visible to others.

## FAQ
**Q: Can I work offline?**
A: Yes, if you have installed the PWA version, viewed data is cached. You cannot save changes until online.

**Q: How do I export my model?**
A: Use the "Export" button in the top toolbar. Formats include CSV, RESQML, and PDF reports.
