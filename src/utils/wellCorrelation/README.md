# Well Correlation Utilities

Helper functions and constants for the Well Correlation Tool.

## Modules

- **constants.js**: Global configuration constants (Layout defaults, View modes, Track types).
- **colorPalettes.js**: Standard geological color maps (Facies colors, Log curve colors).
- **trackConfig.js**: Definitions for default track layouts and validation logic.
- **depthHandler.js**: Utilities for depth unit conversion (Meters <-> Feet) and grid generation.
- **lasParser.js**: (Phase 2) Parser for Log ASCII Standard files.
